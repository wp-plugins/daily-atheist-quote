Places a scrolling quote on atheism anywhere on your site - changes daily from a collection of over 150 quotes.

== Description ==

Daily Atheist Quote places a scrolling (marquee-style) quote anywhere in your site through simple php (can be inserted into theme files: page templates, headers, sidebars, etc.) or through a shortcode (can be put into pages, posts, widgets, etc.)

The atheism-themed quotes rotate daily and are selected from a list of almost 150 quotes from various people.

**Sample Quotes**
Whoever praises him as a god of love does not have a high enough opinion of love itself. ~Friedrich Nietzsche

Pick up the Bible and read it.  More atheists come from this exercise than any other single thing. ~Madalyn Murray O’Hair

== Installation ==


1. Upload `atheist-scroll.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Can the quote scroll be added into a file as php? =

Yes. Simply add the following code into your existing php file:

<?php atheist_scroll(); ?>

= Can the quote scroll be added into a file as a shortcode? =

Yes. Simply add the following shortcode into any page, post, widget, etc.

[atheist_scroll]

= Are there any variables or settings I need to adjust? = 

No. It's as simple as adding the code to place the quote wherever you want.

= Can I suggest quotes to be added? = 

Absolutely! I'd love to hear your favorite quote.
